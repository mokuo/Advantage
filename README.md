## PlantUML

### Requirement

- Java
- Graphviz
- [PlantUML \- Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml) 

### Installation

```zsh
# check
brew list graphviz

# install
brew install graphviz
# or upgrade
brew upgrade graphviz
```
